Message identifier (Name):

ALZREP (AIR LANDING ZONE REPORT)

 

 

Related Documents:

ATP-38
ATP-8

 

 

Purpose:

THE ALZREP PROVIDES INFORMATION ACQUIRED DURING RECONNAISSANCE OPERATIONS ABOUT TERRAIN AND ITS POSSIBLE DEVELOPMENT FOR USE AS AN AIR LANDING ZONE (ALZ).

 

 

Sponsor:

NSA (NAVY) AMPHIBIOUS OPERATIONS WORKING GROUP (AMPHIB OPS WG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
