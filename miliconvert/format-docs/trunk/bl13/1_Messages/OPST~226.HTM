OPSTAT UNIT

 

Message identifier (Name):

OPSTAT UNIT (OPERATIONAL STATUS UNIT)

 

 

Related Documents:

ATP-29 SUP 1

 

 

Purpose:

THE OPSTAT UNIT IS USED TO PROVIDE THE OFFICER-IN-TACTICAL COMMAND (OTC) AND OTHER AUTHORITIES OR UNITS, AS APPROPRIATE, WITH OPERATIONAL AND ADMINISTRATIVE INFORMATION, OR TO REPORT CHANGES TO THIS INFORMATION.

 

 

Sponsor:

NSA (NAVY) MARITIME OPERATIONS WORKING GROUP (MAROPS WG)

 

 

Notes:

STRUCTURED LANGUAGE NOTE:  STATEMENTS CONTAINING "SS"* INDICATE THAT THE STATEMENT APPLIES TO ANY VESSEL TYPE BEGINNING WITH THE CHARACTERS "SS", I.E. ANY TYPE OF MILITARY SUBMARINE.

 

 

Remarks:

None

 

 

Status:

In Service

[1.1] Start of MANDATORY Segment MARITIME UNIT DATA which MAY be repeated (unlimited times).

THIS SEGMENT DETAILS CHARACTERISTICS AND CAPABILITIES OF A VESSEL.

THIS SEGMENT COMPRISES SETS 'VESSDAT' THROUGH 'GENTEXT'('OPERATIONAL DEFECTS SUMMARY'). CARE SHOULD BE TAKEN WHEN COMPLETING THIS SEGMENT TO INCLUDE ALL THE CAPABILITIES THAT ARE RELEVANT TO THE OPERATION OR EXERCISE DURING THE PERIOD SPECIFIED. A UNIT’S CAPABILITIES THAT ARE NOT RELEVANT TO THE OPERATION OR EXERCISE SHOULD BE OMITTED.

VESSELS JOINING A FORCE FOR THE FIRST TIME MUST INCLUDE AS MUCH INFORMATION AS POSSIBLE.
