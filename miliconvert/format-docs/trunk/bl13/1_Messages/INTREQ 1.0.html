INTREQ

 

Message identifier (Name):

INTREQ (INTELLIGENCE REQUEST)

 

 

Related Documents:

STANAG 2149

 

 

Purpose:

THE INTREQ IS USED TO STANDARDIZE THE METHOD BY WHICH MILITARY AUTHORITIES AND FORCES OF NATO NATIONS AND NATO COMMANDS REQUEST INTELLIGENCE INFORMATION.

 

 

Sponsor:

NSA (JSB) JOINT INTELLIGENCE WORKING GROUP (JINTWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service

REQLOC

11

REQUEST LOCATION

PROVIDES INFORMATION OF THE SPECIFIC LOCATION OF INTEREST AND THE SIZE OR SHAPE CHARACTERISTICS OF THE TARGET AND MUST ONLY OCCUR, IF SETS "GENLOC" AND "REQPOS" DO NOT OCCUR.

SET 11 (REQLOC) IN CORRESPONDING [A] SEGMENT 7 (INTELLIGENCE REQUEST SEGEMENT) IS REQUIRED IF SET 10 (GENLOC) IN CORRESPONDING [A] SEGMENT 7 (INTELLIGENCE REQUEST SEGEMENT) DOES NOT OCCUR AND SET 12 (REQPOS) IN CORRESPONDING [A] SEGMENT 7 (INTELLIGENCE REQUEST SEGEMENT) DOES NOT OCCUR, OTHERWISE IT IS PROHIBITED.

FIELD 3 IN SET 11 (REQLOC) IN SEGMENT 7 (INTELLIGENCE REQUEST SEGEMENT) IS REQUIRED IF FIELD 1 IN SET 11 (REQLOC) IN SEGMENT 7 (INTELLIGENCE REQUEST SEGEMENT) EQUALS /RADIUS/ OR /WIDTH/ OR /ELLIPSE/, OTHERWISE IT IS PROHIBITED.

FIELD 3 IN SET 11 (REQLOC) IN SEGMENT 7 (INTELLIGENCE REQUEST SEGEMENT) MUST USE ALTERNATIVE FF2178-002 IF FIELD 1 IN SET 11 (REQLOC) IN SEGMENT 7 (INTELLIGENCE REQUEST SEGEMENT) EQUALS /RADIUS/.

FIELD 3 IN SET 11 (REQLOC) IN SEGMENT 7 (INTELLIGENCE REQUEST SEGEMENT) MUST USE ALTERNATIVE FF2179-002 IF FIELD 1 IN SET 11 (REQLOC) IN SEGMENT 7 (INTELLIGENCE REQUEST SEGEMENT) EQUALS /WIDTH/.

FIELD 3 IN SET 11 (REQLOC) IN SEGMENT 7 (INTELLIGENCE REQUEST SEGEMENT) MUST USE ALTERNATIVE FF2180-002 IF FIELD 1 IN SET 11 (REQLOC) IN SEGMENT 7 (INTELLIGENCE REQUEST SEGEMENT) EQUALS /ELLIPSE/.

SIGNIFICANT FIELD 6 IN EVERY OCCURRENCE OF FIELD GROUP IN SET 11 (REQLOC) IN SEGMENT 7 (INTELLIGENCE REQUEST SEGEMENT) IS REQUIRED IF FIELD 1 IN SET 11 (REQLOC) IN SEGMENT 7 (INTELLIGENCE REQUEST SEGEMENT) EQUALS /MULTIPLE/ OR /LINE/ OR /TRACEPNT/ OR /WIDTH/, OTHERWISE IT IS PROHIBITED.
