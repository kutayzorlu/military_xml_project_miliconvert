LOCATOR

 

Message identifier (Name):

LOCATOR (MARITIME FORCE LOCATOR)

 

 

Related Documents:

ATP-1 VOL I

 

 

Purpose:

THE LOCATOR IS USED TO REPORT SURFACE, SUBSURFACE, AIR, MINES, OR SPECIAL INTEREST UNITS OPERATING IN THE MARITIME ENVIRONMENT.

 

 

Sponsor:

NSA (NAVY) MARITIME OPERATIONS WORKING GROUP (MAROPS WG)

 

 

Notes:

THE SETS HEADING (MULTIPLE CONTACTS) THROUGH DESPLACE FORM AN OPERATIONALLY DETERMINED SEGMENT TO REPORT MULTIPLE (STACKED) CONTACTS WITH SINGLE TIME AND POSITION AMPLIFYING DATA.

 

 

Remarks:

None

 

 

Status:

In Service
